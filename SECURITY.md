# Security Policy

This repository contains demo code only — it is not a published package.
Vulnerabilities in the demos themselves are unlikely to be impactful, but
if you find one, please disclose it privately so we can fix it before
publishing details.

## Reporting a Vulnerability

Please report vulnerabilities privately to **security@trulayer.ai**.

- Include a clear description, reproduction steps, and the example file affected.
- Do **not** open a public GitHub issue for security reports.
- We aim to acknowledge within 5 business days.

For vulnerabilities in the underlying SDK, please report them on the
[`client-typescript`](https://github.com/trulayer/client-typescript) repository instead.
